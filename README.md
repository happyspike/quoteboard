### Developer Setup
##### Requirements
- `gem install sass`
- `npm install -g riot`

For js:
`riot --watch public/src/js/ public/dist/js`

For css:
`- sass --watch public/src/css/:public/dist/css --default-encoding UTF-8 --scss`
